### 정리
> 사실 내용 자체는 쉬운 css로만 구성이 되어있다. 


```css
    appearance:none
```
+ 브라우저가 기본적으로 가지고 있는 테마를 숨길 수 있게 해준다.
+ 여기서는 input:checkbox의 기본 크롬 css를 안보이게 해준다.

```
    diagonal length of square
```
+ 정사각형의 대각선 길이를 구해서 그림자를 적용함
